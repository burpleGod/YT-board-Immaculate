**Antlom's Gaming Dashboard**
A specialized content management system and immersive dashboard built for YouTube roleplay creators. This application integrates production tracking with in-character world-building tools, utilizing a React-based architecture and persistent local storage.

**Features**
Production Management
YouTube Pipeline: Track episode status from planning to upload with metadata support for thumbnails and scheduled dates.

Idea Ledger: Categorized board for character concepts, roleplay rules, and quest ideas.

The Forge: Real-time UI customization engine for adjusting accent colors, background modes, and font settings.

Immersive Tools
Roleplay Journal: Handwritten-style entry system with support for multiple typography sets including Kalam and EB Garamond.

Gallery System: Localized image storage for screenshots and concept art with category filtering.

Atmospheric UI: Dynamic CSS-driven environmental effects including runic glows and parallax fog drifting.
